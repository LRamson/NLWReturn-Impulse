## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- Web:
  - React
  - Typescript
  - Tailwindcss
  - Phosphor Icons
- Server:
  - Express
  - Typescript
  - Primsa
  - Jest

## Projeto

Este projeto trata-se de um widget colhetor de feedbacks. Feito para ser responsivo e utilizável em qualquer website. Projeto não finalizado.
